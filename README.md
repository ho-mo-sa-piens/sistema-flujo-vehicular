# Sistema de Flujo Vehicular — Avenida X

Trabajo práctico de análisis sistémico. Visualización web del **Sistema de Flujo Vehicular de la Avenida X** aplicando conceptos de pensamiento sistémico.

## 🗂️ Estructura del proyecto

```
sistema-flujo-vehicular/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos de la interfaz
├── js/
│   ├── data.js         # Datos del sistema (fronteras, componentes, etc.)
│   └── render.js       # Lógica de renderizado dinámico
└── README.md
```

## 🚀 Cómo correr el proyecto

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/TU_USUARIO/sistema-flujo-vehicular.git
   ```
2. Abrir `index.html` en el navegador (no requiere servidor).

## 📋 Contenido del análisis

| Sección | Descripción |
|---|---|
| Propósito | Objetivo del sistema |
| Fronteras | Qué está dentro y fuera del sistema |
| Componentes | Elementos clave |
| Entradas / Salidas | Inputs y outputs del sistema |
| Retroalimentación | Loops negativos y positivos |
| Entorno | Factores externos |
| Interacciones | Relaciones entre componentes |
